- Instant screenshots — full screen, specific windows, or custom regions
- 4 MCP tools that Copilot Chat agents can invoke autonomously
- Smart key frame extraction for AI analysis
- Full local, full private — all processing stays on your machine
- Process triggers — auto-record when a specific process starts
- Event triggers — auto-record on debug sessions, tasks, or test runs
- Stability detection — auto-stop when screen becomes visually stable
- Multiple export formats: MP4, WebM, and GIF
